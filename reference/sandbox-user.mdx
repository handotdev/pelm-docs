---
title: "Sandbox User"
description: "If you aren't ready to use real data yet or don't have a utility account available to test with, you can test Pelm with the Sandbox User."
---

In Connect, select the utility "Pacific Gas and Electric".

On the credentials screen, enter the following credentials:  
username: `user@pelm.com`  
password: `password`

Once you complete the Connect flow, you'll be able to make requests for the Sandbox User just like how you would for any other user.

If you want to skip the Connect flow and start requesting the Sandbox User's data, you can use the following `access_token`. For any request, add the Header `Authorization: Bearer <ACCESS_TOKEN>`.
