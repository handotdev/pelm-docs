## Usage units

| unit\_code | unit  | description                                                 |
| ---------- | ----- | ----------------------------------------------------------- |
| kwh        | kWh   | kilowatt-hour                                               |
| mwh        | MWh   | megawatt-hour, which is equivalent to 1000 \* kilowatt-hour |
| therm      | therm | 100,000 BTUs                                                |

## Emissions units

| unit\_code         | unit          | description                                           |
| ------------------ | ------------- | ----------------------------------------------------- |
| kg\_co2e\_per\_kwh | kg CO2e / kWh | kilograms of CO2 equivalent emitted per kilowatt-hour |
| kg\_co2            | kg CO2e       | kilograms of CO2 equivalent                           |
